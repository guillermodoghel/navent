# Backend
------------

``cd backend/``

### Configuracion
Corer  con maven

``mvn clean install``

Correr desde eclipse, deployar el war en un tomcat, o como se guste

# Frontend
------------

``cd frontend/``

### Configuracion

Corer  con node

``npm install``

para correr en modo debug

``npm start``

para ejecutar en build

compilar
``npm run build``

correr
``serve -s build``
